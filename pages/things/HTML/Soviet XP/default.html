苏维埃 XP


关于苏维埃 XP

采用微软原版XP SP3镜像制作，能通过正版验证，安全无毒，没有捆绑。（修改过程中只有我一个人参加，看在我这么刻苦的份上，你就不打算去我的B站主页支持一下我吗QAQ）

这个系统的故事

在1990年，苏联一个老头发明了这个系统。
后来，1991年苏联解体，这个系统的开发也停止了。
再后来，这个系统落在了比尔盖茨的手里。
微软稍作修改，便将这个系统发布成了Windows XP
所以我们要支持正版，抵制微软[doge]
最后，最重要的是：上面的一切都是扯淡（话说你不觉得这个故事好像在哪里听过吗啊喂?)

 

注意：1.安装过程中出现字体缺失系正常现象，是由于过度精简安装程序导致的，不影响安装完成后系统的正常使用!

2.安装时最好不要使用VM的简易安装(关于如何跳过简易安装），否则会导致无人值守失效从而要求输入激活码。
